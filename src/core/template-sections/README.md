# Page Sections

Reusable, self-contained page section components.

## Purpose
- Build blocks for composing complete pages
- JSON-configurable via props
- Theme-aware using design tokens
- Registered in PageSectionRegistry

## Categories
- **Hero** - Above-the-fold sections
- **Features** - Feature showcases
- **Pricing** - Pricing tables
- **Testimonials** - Customer reviews
- **CTA** - Call-to-action sections
- **Footer** - Page footers
- **Header** - Navigation bars
- **Content** - Text/image content blocks
- **Gallery** - Image/video galleries
- **Team** - Team member displays
- **Blog** - Blog post lists
- **Contact** - Contact forms
- **FAQ** - FAQ accordions
- **Stats** - Statistics displays
- **Logos** - Logo showcases
