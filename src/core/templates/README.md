# Website Templates

Complete, multi-page website templates composed of sections.

## Purpose
- Production-ready website configurations
- JSON-based page definitions
- Pre-configured navigation and footer
- SEO metadata included
- Tenant-customizable

## Categories
- **landing-page** - Single-page landing sites
- **saas** - SaaS product websites
- **portfolio** - Portfolio/showcase sites
- **blog** - Blog/content sites
- **ecommerce** - Online store templates
- **corporate** - Corporate websites
- **agency** - Agency/studio sites
- **personal** - Personal brand sites
- **education** - Educational platforms
- **nonprofit** - Nonprofit organizations
